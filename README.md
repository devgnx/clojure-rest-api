# Clojure REST API

A simple Clojure boilerplate for REST API Apps

### Requirements:  
- Lein v1.0.0 https://leiningen.org
- CURL https://curl.haxx.se/

___

### Usage:

#### Run App
```sh
lein run
```

#### API: Ping/Pong route
```
curl -s localhost:8080/api/ping
```

___

### Testing

#### Run auto-tests (dev environment only)
```sh
lein auto-test
```
#### Run tests
```sh
lein test
```
